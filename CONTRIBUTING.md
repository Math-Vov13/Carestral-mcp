# HOW TO CONTRIBUTE

## Commands

### Project Dependencies

Sync project dep
```sh
uv sync
```

install package
```sh
uv add <package-name>
```

install dev package
```sh
uv add --dev <package-name>
```


### MCP Server

Run MCP server
```sh
fastmcp run src/server.py
```

### Linter && formater

```sh
ruff
```

### Unit Tests

```sh
pytest
```
